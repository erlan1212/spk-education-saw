# SPK Education - Metode SAW

Sistem Pendukung Keputusan pemilihan platform dan course education terbaik menggunakan metode SAW (Simple Additive Weighting).

## Features
- Data Cleaning
- Normalisasi SAW
- Perhitungan SAW
- Ranking Platform
- Ranking Course berdasarkan kategori
- Visualisasi Data
- Validasi Bobot & Dataset

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## Run
```bash
streamlit run main.py
